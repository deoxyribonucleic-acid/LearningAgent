# Practice: Chapter 1.3: Implementing a 5-Minute Agent

## Usage
Copy the `.env.example` file to `.env` and fill in your API keys for OpenAI and Tavily services.

```bash
cp .env_template .env
```
Then, run the Jupyter Notebook `Chapter1-3-Implementing-a-5-Minute-Agent.ipynb` to see the implementation of a simple agent that can perform tasks using the OpenAI API and Tavily API.
